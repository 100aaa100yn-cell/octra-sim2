オクトラ大陸の覇者 最大ダメージシミュレーター v0.3.0

【起動方法】
1. ZIPを展開します。
2. index.htmlをダブルクリックします。

【v0.3.0追加内容】
・極召喚：バハムートを実データ形式へ更新
・敵全体への火属性4回攻撃（威力105）
・ブースト威力 115 / 150 / 200
・火、氷、雷、風、光、闇弱点を突ける表示
・ダメージ計算属性は火として処理
・完全回避、完全防御を無視する効果を表示
・BP MAXかつブレイク条件成立時、同じ攻撃を再発動
・最初の4連撃でシールドを削り切った場合も再発動
・敵シールド入力欄を追加
・本体4ヒットと再発動4ヒットを分けて表示

【注意】
ダメージ計算式やキャラクター基礎値は、現段階では検証用の仮式・仮値を含みます。
シールドは弱点時に1ヒットあたり1減少する簡易方式です。
