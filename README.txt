オクトラ大陸の覇者 最大ダメージシミュレーター v0.2.0

【起動方法】
1. ZIPを展開します。
2. index.html をダブルクリックします。
3. ブラウザ上でそのまま使用できます。

【v0.2.0の追加機能】
・キャラクター選択時に固有サポート効果を表示
・技選択時にアビリティ追加効果を表示
・物攻／属攻アップを自動加算
・武器／属性ダメージアップを自動加算
・物防／属防ダウンを自動加算
・ダメージ上限アップを自動加算
・手動入力値と自動効果の内訳を計算結果に表示
・「行動後に付与」の効果を今回の攻撃へ仮適用する切替を追加

【効果タイミング】
通常、「行動後に付与」の効果は、その技自身のダメージには適用されません。
次の攻撃時の状態を比較したい場合のみ、画面のチェックをONにしてください。

【重要】
現在のキャラクター数値、アビリティ威力、効果内容、計算式は初期動作確認用の仮データを含みます。
実データ化するときは data/characters.js と data/skills.js を5人ずつ更新してください。
