オクトラ大陸の覇者 最大ダメージシミュレーター v0.4.0

【起動方法】
1. ZIPを展開します。
2. index.htmlをダブルクリックします。

【v0.4.0追加内容】
・選択アビリティのBP別継続ターン表示を追加
・現在選択中のBPに対応する継続ターンを強調表示
・BP0 / BP1 / BP2 / BP MAXの威力比較表を追加
・効果ごとに対象、発動タイミング、継続ターンを表示
・即時効果、常時効果、条件効果は「継続ターンなし」と表示
・アビリティ詳細パネルを追加
・攻撃種別、対象、属性、ヒット数、弱点判定を一覧表示
・バハムートの再発動条件と無視効果を詳細パネルに表示

【v0.3.0から継続している機能】
・極召喚：バハムートの火属性4回攻撃
・ブースト威力 115 / 150 / 200
・火、氷、雷、風、光、闇弱点を突ける表示
・完全回避、完全防御を無視
・BP MAXかつブレイク条件成立時に同じ攻撃を再発動
・最初の4連撃でシールドを削り切った場合も再発動

【注意】
継続ターンは現段階の検証用データを含みます。今後、Game8の記載を確認しながらキャラクター5人ずつ実データへ置き換えます。
ダメージ計算式やキャラクター基礎値にも、現段階では検証用の仮式・仮値を含みます。
