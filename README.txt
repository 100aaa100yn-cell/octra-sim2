オクトラ大陸の覇者 最大ダメージシミュレーター v0.4.1

【更新方法】
ZIPの解凍は不要です。
今回添付する各ファイルを、既存プロジェクト内の同名ファイルへ上書きしてください。

【上書き対象】
・index.html
・README.txt
・data/characters.js
・data/skills.js
・js/app.js
・js/damage-calculator.js

【v0.4.1修正内容】
・誤って「極召喚：バハムート」としていた攻撃名を「アルテマ」に修正
・ユウナのバトルアビリティ10種を登録
・消費SPをアビリティ詳細に表示
・攻撃、回復、補助アビリティを区別
・BP別威力、回復効力、最大HP割合回復量を表示
・BP別の効果継続ターンを表示
・BP MAX時の対象拡張を表示
・ヴァルファーレ、イクシオン、アニマの召喚獣・追撃情報を表示
・アルテマの多属性弱点判定、完全回避・完全防御無視、再発動条件を表示
・アニマの固有ダメージ上限+200,000を計算へ反映
・回復／補助アビリティをダメージ計算した場合は案内を表示

【注意】
召喚獣追撃は今回、詳細表示用データまで登録しています。
追撃込みのターン進行シミュレーションと召喚獣状態の保持は次段階で実装します。
