オクトラ大陸の覇者 最大ダメージシミュレーター v0.1.0

【起動方法】
1. ZIPを展開します。
2. index.html をダブルクリックします。
3. ブラウザ上でそのまま使用できます。

【現在できること】
・8人編成のキャラクター切り替え
・キャラクターごとのアビリティ選択
・ブースト0～MAXの切り替え
・物攻／属攻補正
・攻撃バフ、ダメージアップ、上限加算
・敵の物防／属防、防御デバフ
・弱点、ブレイク、乱数最大値
・1ヒットごとのダメージ上限判定
・選択キャラの最大アビリティ自動検索
・同条件での理論撃破回数表示

【重要】
現在のキャラクター数値、アビリティ威力、計算式は初期動作確認用の仮データを含みます。
今後、Game8等で確認した実データへ5人ずつ置き換えてください。
実ゲームの正式なダメージ式が確定した場合は、
js/damage-calculator.js の「初期版の仮計算式」の部分を差し替えるだけで対応できます。
